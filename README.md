# StockSphere

StockSphere is a simple Inventory Management System built using Spring Boot and MySQL. It allows users to manage product inventory through a clean web interface.

## Features

- Add a new product
- View all products
- Update product details
- Delete products
- Store inventory data in MySQL

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript
- Maven
- Git
- GitHub

## Project Structure

- **Backend:** Spring Boot REST API
- **Frontend:** HTML, CSS and JavaScript
- **Database:** MySQL
